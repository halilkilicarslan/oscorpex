3. **auth-routes tenant check** -- API key silmede tenant_id dogrulama ekle
4. **tenant-repo transaction** -- createTenantWithOwner'i withTransaction icine al
5. **sandbox realpath** -- checkPathAllowed'a realpathSync ekle

6. **lastFailureClassification** -- Provider hata sinifini error objesine ekle, task-executor'da kullan

9. **approval timeout** -- waiting_approval gecisinde startedAt veya approvalRequestedAt set et
10. **budget guard atomicity** -- Tek transaction icinde SUM + limit kontrol
